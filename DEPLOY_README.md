## 🎯 Em 2 Minutos

```bash
1. Crie conta GitHub (github.com/signup)
2. Push seu código
3. Deploy backend no Render (render.com)
4. Deploy frontend no Vercel (vercel.com)
5. Compartilhe o link!
```

---

## ⚠️ Limitações Free Tier

### Render
- Sleep após 15 min sem requisição (30s para acordar)
- 0.5GB RAM
- Sem garantia de uptime 99.9%

### Vercel
- 12 deploys/mês com git auto-deploy

**Solução**: Clique em "Redeploy" se precisar fazer mais deploys

---

## 📞 Problema Comum

**"504 Gateway Timeout"**
- Backend está dormindo
- Solução: Espere 30s e tente de novo
